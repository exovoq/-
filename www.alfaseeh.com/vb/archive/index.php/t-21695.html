مكتبة أسامة بحثت فيها عن محاضرات فأخرج لي أوراقا كثيرة لكن واحدة منها لم تكن في الأدب كلها لقسم النحو أو دكتوراه أدب
أما الجدول :

السبت : منهج البحث وتحقيق التراث ( أ.د/محمود توفيق) +أدب ( د. صالح بدوي )
الأحد : أدب الدعوة ( أ.د/ عبد الله الزهراني ) + نقد ( أ.د/ محمد مريسي )
الاثنين : بلاغة ( أ.د/محمود توفيق)
الثلاثاء : أدب ( أ.د/ عبد الله العضيبي )
الأربعاء : نقد ( كان أ.د / صالح الزهراني وأخبرنا الدكتور المريسي أنه سيكون معنا كذلك الأربعاء ) + بلاغة ( أ.د/ محمد أبو موسى )

جميع مواعيد الجدول : المادة الأولى : 1، 2 والمادة الثانية : 4.3
