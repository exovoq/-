Шгу Чссх ЧсбЭуф ЧсбЭэу

оЧс Чссх Чскйэу нэ пЪЧШх ЧсЭпэу : (нХаЧ фоб нэ ЧсфЧоцб*насп эцуЦа эцу кгэб* ксь ЧспЧнбэф лэб эгэб)

апб ЧсксЧуЩ ЧсвуЮдбэ нэ Ънгэбх (ЧспдЧн) цЬхЧ нэ УкбЧШ псуЩ (эцуЦа) спфэ су Унхух Шдпс ЬэЯ....цхаЧ фе псЧух :нХф осЪ : Шу ЧфЪеШ ХаЧ Ё цпэн еЭ Уф эок { эѓцњуѓЦіађ } йбнЧ№ сэцу кгэбП осЪ : ЧфЪеШ ХаЧ ШуЧ Ясј ксэх ЧсЬвЧС Ё сУфј Чсукфь : нХаЧ фоб нэ ЧсфЧоцб кгб ЧсУуб ксь ЧспЧнбэф Ё цЧсаэ УЬЧв цоцк { эѓцњуѓЦіађ } йбнЧ№ сэцу кгэб : Уфј Чсукфь : насп цоЪ Чсфоб цоцк эцу кгэб Ё сУфј эцу ЧсоэЧуЩ эУЪэ цэок Ээф эфоб нэ ЧсфЧоцб .

цоЧс ЧсЧсцгэ нэ Ънгэбх бцЭ ЧсукЧфэ :цЬцв Уф эпцф эцуЦа йбнЧ№ угЪобЧ№ сэцу кгэб Уэ енЩ сх нсуЧ ЪоЯу ксэх еЧб ЭЧсЧ уфх цЧсаэ УЬЧв асп ксь уЧ нэ ЧспдЧн Уф Чсукфь насп цоЪ Чсфоб цоцк эцу кгэб сУф эцу ЧсоэЧуЩ эУЪэ цэок Ээф эфоб нэ ЧсфЧоцб нхц ксь уфцЧс вуф ЧсбШэк ЧскэЯ нэх Уэ цоцк ЧскэЯ нэх цуЧ сх насп Чсцоцк цоцк эцу ЧсЮ цууЧ апб эксу ХфЯнЧк уЧ эЪцху уф ЪоЯэу укуцс ЧсуеЯб Уц укуцс уЧ нэ есЪх ксь ЧсуеЯбЧф Ькс йбн Чсцоцк ЧсуоЯб Уц йбн кгэб цЧсЪебэЭ Шснй цоцк ХШбЧв ссукфь цЪоеэ кф Ькс ЧсвуЧф уйбцн ЧсвуЧф ШбЬцкх Хсь ЧсЭЯЫ нЪЯШб
