أليست / ليتك تستقم فتعش سعيداً ( من اقتران جواب الشرط بالفاء )


قاعدة اقتران جواب الشرط بالفاء :-
يجب اقتران جواب الشرط بفاء تربط جملته بفعل الشرط إذا لم يصلح
جواب الشرط للجزم في المواضع الآتية :-
1- إذا كان الجواب جملة اسمية مثال :- من ظُلمَ فالله نصيرهـ 0
2- ======== طلبية مثال :- إن خفي على أحدكم أمر فليسأل 0
3- ==== فعلا جامدا :- ليس - عسى - نعم - بئس
مثال :- حيثما تجد العيش سهلا فانعم به
3- إذا كان فعلا منفيا بلن 0

مثال :- ماتتعلم في الصغر فلن تنساهـ في الكبر 0
4- إذا كان فعلا مقرونا بقد 0
( من يُطعْ الرسولَ فقد أطاعَ اللهَ )
5- إذا كان فعلا مضارعا مقترنا بالسين أو سوف مثال لسوف :- إن ظلمت فسوف تحاسب 0

ليتك تستقيم تعش سعيدًا


حكم الفعل المضارع الواقع في جواب الطلب الجزم

كأن يقع بعد أمر أو نهي أو تمني أو ترجٍ أو استفهام أو عرض أو تحضيض
والعرض يكون بـ ( ألا )
والتحضيض بــ ( هــلا ) اللام عليها شدة 0
إعراب ألا في المثال الثاني :-
ألا حرف عرض مبني على السكون لامحل له من الإعراب 0
