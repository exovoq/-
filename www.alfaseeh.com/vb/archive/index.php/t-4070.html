ΥέΝΗΚ ΚΗΡνΞν ΚΠζΘ αγΗ ΘεάΗ = ίγΟΗ, ΚυάΡμ ΓήΡΓΚάγυ ΥέΝΗΚάνΏ
ΓζσανΣ νΦΨΡγ ΗαΕΘάΗΑ ΘΓΝΡέάν= ΓζσαΗ νυΨαψ ΗαήεάΡ γάδ ίαγΗΚάνΏ
γΛα ΗαΣνζα γΟΗγΪν ζΗ ΝΡήΚάν! = ΓΚΡμ ΣΚζήάΩ δυζψγάΗ ΟγΪΗΚάνΏ
ΓζσγΗ ΣγΪΚγ ΥΡΞΚν ΗαΝΡψμ ΘίγΏ = ζΗ αεέΚάμ! ΓΓαέΚάγ ΥΡΞΗΚάνΏ
ΓδΗ γδ δΤάΓΚ ΓΘνψάΙ ΤγψάΗΑ αάγ = ΓΡΦσ ΗαεζΗδ, ζαγ ΚΫ ΞΨζΗΚν
ΓδΗ ΓγΙ ΗαήάΡΒδ αΗ ΓΡΦάμ Θάε = ΘΟαΗ, ζγδε ήΟ ΗΣΚήνάΚυ ΥέΗΚάν
ΓδΗ γδ δΤΓΚ Ϊαμ ΗαΜεΗΟ ΓδΗ ΗαΚν = ΓΘαάμ ΘάαΗΑ ΗαέΗΚΝνάδ ΓυΘάΗΚάν
ΓδΗ γδ ΣγζΚυ Ϊδ ΗαΟδΗνάΗ ΡέΪάΙ = ΓδΗ ΓγΙ ΗαΕΣάαΗγ ζΗ ΝΣσΡΗΚάν!!
νΗ γδ ΚΡζδ ΓγΗγίγ ΜΡΝν ΓγάΗ = αΚγ ΓυΣΗΡμ ΗαάΠα ζΗαΤεάζΗΚΏ
Γζ ΚΨγΪζδ ΘδΥΡΚάν ΘίαΗγίάγΏ = γΗ ΘΗα δΥΡΚίγ ΣάαΗΝ ΪφΟΗΚάνΏ
ΓΘσδνψ νΗ γδ ίδΚάγυ ΞνάΡ ΗαάζΡμ = γΗΠΗ ΟεΗίγ νάΗ Πζν ΗαΪσγάΗΚΏ
γΗ ΘΗα ΓΡΦν ΚυΣΚΘάΗΝ ΓγΗγίάγ = ζΚγάζΚ Βδ ήΨΗέεάΗ ΛγάΡΗΚάν
ΓυΣΘμ ζΚδεΤδν ΗαΠΖΗΘ ΦζΗΡνςάΗ = ζΘΞΣΙ, ίάγ νυΫΚΥΘάδ ΘδΗΚάν!
ζνγζΚ ΓΨέΗαν ζΓΜΡΪ ΫΥΚάν = ζνΩαψ νΘίνεάγ ΛάΡμ ΪΡΥΗΚάν
ζνγζΚ ΤφνΘρ έν ΝΥΗΟ ΫφαΗαεάγ = ΘήΠΗΖάέ ΚάΚάΡμ ζ ΜάΡψΗέάΗΚφ
ΓΘίνεάγυ ζΝάΟν, ζΓδΚάγ εΗεδάΗ = αΗ ΚΟΡίάζδ ΝήνήάΙ ΗαγάΓΣάΗΙ
ΚΣΚγΚΪάζδ ΘζήΚίάγ ζΝνΗΚάίάγ = αΗενδ Ϊδ ΜΡΝν ζΪδ ίΘζΗΚάν
ΓΘσδνψ νΗ γδ ήάΟ ίΣΚίάγ ΘΡΟΚάν = νΗ γάδ ήάζΗγ ΝνΗΚίάγ ΞνΡΗΚάν
ΓζανΣ γδίγ γδ νΫΗΡ Ϊαάνψ γάδ = Ωαγ ΓΥΗΘ ΗαΪγή γδ ΝΡγΗΚάνΏ
Γζαγ ΓΡΘψ ΓΤΗζΣάΗ γάδ ήΘαίάγ = ίΗδάζΗ αΓγΜάΗΟν ΓΘάΡψ ΘυάδάΗΙφΏ
Γζαγ ΓΣυΟϊ ΪΡΤ ΗαΛΡνψάΗ νάζγ Γδ= ΓΣήμ ΗαζΫμ Ογ ήΗΟΚνφ ζΝυγΗΚάνφ
