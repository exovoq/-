лвЩ Шэф фЧб цкЧб

уЭуцЯ ЧсфЬЧб


уф умцЬ ШЭмбпѓ ЪШмвлѕ ЧсУоумЧбѕ = эЧ дЧиЦмЧ фмЧСЪ Шмх ЧсУЮимЧбѕ

эЧ цЧонЧ нэ вЭмн бцЭмпѓ уЧЫмсЧ = ЪЧбэмЮѓ кмвјђ лмЧсмх ЧсУдммбЧб

эЧ дЧиЦмЧ гоимЪњ ксмь УоЯЧумх = слмЩѕ ЧсЮэЧфмЩ цЧсхмць ЧсмлмЯЧб

УфЧ сЧ Укэап умф ЮгЧгмЩ ЭЧомЯ = Шс умф Шфэмп куЧхму ЧсмЯцсЧб

уф уЯкэ цес цуЧ цесмцЧ Хсмь = УлцЧб бцЭпѓ Шмс ксэмп УлмЧбцЧ

эЧ дЧиЦЧ Уцхь ЧЭЪумЧсЧЪі Чсфхмь = сујЧ ЧфЪШхЪѓ цнэ еумцЯп ЭмЧбцЧ

эЧ УэхЧ Чсуикцф нэ Чсйхб ЧгЪць = нмэ Чсмас умф кмЧЯЧп цЧспнмЧб

эгкцф нэп ШмУф ЪШэмк пШэкіхму = УсЧ ЪЭмгјѓ ШумЧ эмомцс ЧсмкмЧб

Ъбпцп ссЬцк ЧсукбШЯ нмэ ЧсЭдмЧ = оикмцп уфмп кеЧШмЩё УдммбЧб

Уц сгЪ уф пШЯ ЧсШсЧЯ УсгмЪ нмэ = дбэЧфхмЧ Янмоѕ ЧсммЯуі ЧсмумцЧбѕ

Уц сэг УинЧс (ЧсобЧбЩ) ху хумц = цШфмц (ЧсбумЧс) ЧснЪэмЩѕ ЧсУЭмбЧб

ц(укгпб ЧсдЧиэ) Чсаэ су эЮЪжШњ = ШмЯумЧх ХсЧ ЧсемэмЯѕ цЧсУЭммбЧб

Уц су эбцЧ Яукѓ ЧселЧб цЬцкѓхму = цксэсѕхму жЧомЪ Шмх ЧсУуемЧб

УцЧх умф Ьѓмцњб ЧсобэмШ цЬЯЪјмхѕ = фмЧбЧ экбШмЯ нцомхмЧ ХкмемЧб

су эксуцЧ лжШ ЧскѕШЧШ цсмц ЯѓбѓцЧ = уЧ пмЧф ХсЧ нмэ ЧсухЧфмЩ лмЧбцЧ

нпУфхму пнмбцЧ ШмЯэмф уЭмумЯђ = цуф ЧскбцШмЩ УнсЪмцЧ цЧфхмЧбцЧ
