ذهب الرجل نفسه إلى العمل

الرجل : متبوع وهو المؤكَّد ( اسم مفعول )
نفسه : تابع وهو المؤكِّد ( اسم فاعل )

إذن التوكيد هو المؤكَّد بفتح الكاف وتشديدها

فهل هذا الكلام صحيحٌ أم فهمي خاطئ؟؟
