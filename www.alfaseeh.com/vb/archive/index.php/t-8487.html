УЮиЧС дЧЦкЩ ( сЧ Ъос...ц ос)

--------------------------------------------------------------------------------

Шгу Чссх ЧсбЭуф ЧсбЭэу

УоЯу спу - УЮцЧфэ ц УЮцЧЪэ - хах ЧсудЧбпЩ ЧсШгэиЩ ЧсЪэ ЧфЪоэЪхЧ уф пЪШ ( уцгцкЩ ЧсХусЧС ц ЧсХкбЧШ ) ЧсэнЧфЧ ЧсцЧбЯэфэ Я. уЭуЯ ЭуцЯ и1 1998 ЯЧб Чснпб ЧссШфЧфэ .

- сЧ Ъос " хах ХфгЧфЩ "... ц ос " хаЧ ХфгЧф " (ЧсХфгЧф эок ксь Чсапб ц ЧсУфЫь )
- сЧ Ъос " эЬШ Уф ЪЪцнб нэх ЧсЮШбЩ "... ц ос " эЬШ Уф ЪЪцЧнб нэх ЧсЮШбЩ ".
- сЧ Ъос " УЫјѓѓб ксэх "... ц ос " УЫјб нэх " Уц " Шх " .
- сЧ Ъос" нсЧф цЧЫо уф псЧух "... ц ос " нсЧф цЧЫо ШпсЧух " (Чснкс эЪкЯь ШЧсШЧС)
- сЧ Ъос " скШѓ ЯцбЧ "... ц ос "уЫјс ЯцбЧ " Уц " УЯјь ЯцбЧ " (нкс "скШ" сЧву сЧ эЪкЯь Хсь ункс Шх)
- сЧ Ъос" кЧд ЪЬбШЩ " ... ц ос " кЧфь ЪЬбШЩ " (нкс "кЧд" сЧву сЧ эЪкЯь Хсь ункс Шх)

- сЧ Ъос" ХЭЪѓжб нсЧф " ... ц ос" УѕЭЪжб нсЧф "- Шжу ЧсЪЧС ( хаЧ Чснкс эгЪЮЯу ЯЧЦуЧ нэ ЧсуЬхцс)
- сЧ Ъос" сЧ гэјуЧ ц Уф " ... ц ос " ц сЧ гэјуЧ Уф " (цЧц ЧсХкЪбЧжэЩ Ъпцф нэ Уцс ЧсЬусЩ сЧ нэ цгихЧ)
- сЧ Ъос " хах кЧЦсЪх " ... ц ос " хах УгбЪх " (ЧскЧЦсЩ уФфЫ ЧскЧЦс)
- сЧ Ъос " Уогу ШУф энкс " ... ц ос " Уогу ксь Уф энкс "(ЧсШЧС ЪЯЮс ксь Чсуогу Шх ц сэг ксь ЬцЧШ Чсогу)
- сЧ Ъос " бжЮ ссУуб " ... ц ос " Уакф ссУуб " (бжЮ Шукфь пгб ЧсдэС ЧсэЧШг Уц Укиь осэсЧ)
- сЧ Ъос " УЯуф ксь ЧсдэС " ... ц ос " УЯуф ЧсдэС " (нкс "УЯуф уЪкЯ Шлэб Эбн Ьб)
- сЧ Ъос " всЧс ЧсШэж " ... ц ос " ЧсТЭ " (снйЩ всЧс сЧ Ъисо ксь ЧсШэж)
- сЧ Ъос " ЧсбнЧЪ ЧсШЧсэЩ " ... ц ос " ЧсбнЧЪ ЧсШЧсэ " (ЧсбнЧЪ уапјб)
-сЧ Ъос " бд ЧсусЭ ксь ЧсикЧу " ... ц ос " абј ЧсусЭ ксь ЧсикЧу " (Чсбд ЮЧе ШЧсуЧС ц фЭцх уф ЧсуЧЦкЧЪ


- сф Уоцс спу " цЯЧкЧ " ц спф Хсь " соЧС ТЮб" нэ фнг Чсуцжцк
