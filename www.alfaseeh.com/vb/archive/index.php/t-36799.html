ц уѕгмЧсмумэмф усмцпѓ бцуЧ ШмкмЯумЧ = усмјѓмпмЪмуѕмцЧ нмімэммфммѓЧ уѕсмцпѓ ЧсмЬѓмЧф

ШЧсУуг гмЧбЪ ЮмэсмѕфмЧ ц гэмцнмѕфмЧ = ЯпммјѓЪ омімсЧк Чснммѕбг ц ЧсмбѕцуЧф

ц Чсэмцу нмэ умѓЯбэЯ Ъмѓелмб УумЪэ = ц Ъмгмэммбѕ ЮЧжімкмѓмЩ№ ШміммсЧ УкмцЧф

дммЪмјѓЧф Шэмф ЧсУуг ц Чсмэмцу Чсмаэ = жмѓмЧоммЪ Шмх ЧсУЭсЧу ШЧснмімЪмэммѓЧф

дммЪмјѓЧф эмѓЧ цимфэ Чсаэ оммѓЯ бѓЧкмѓх = дммцоё эмШмэмЪ ШмумомсмЪэ ц ЬѓмфЧфэ

дмкмШё эмѕкмЧоммб пУгмѓх умѓумЬмѕцЬЩ№ = Уумгмь экммЧфмэ дммхммцЩ ЧсХЯуммЧф

эЬммбэ цбЧС ЭмШмэммШммЩђ лммбШмэмЩђ = Ъмѕгмомэмх пмУг Чсмас ц ЧсЮѕмгммбЧф

хмѓЬѓбЪ пмЪЧШ Чссх ЭмЪь УемШЭЪ = умЫмс ЧссмјѓмомэммимЩ нэ Шмфэ кімсумѓЧф

пмммсё эЬмммб ЫммэммЧШммхмЧ сіумѓмфмѓЧуміммх = лмѓмЬмѓмбэмјѓЩ ЪммѕгмШмь ШмсЧ ЧгмЪміЦмаЧф

ЬѓбіШмѓЪ ц ШмѓЧф кѕцЧбхЧ ц ЮѕцЧбхмЧ = ЪмѓбЬц хмімсЧс ЧскмэмЯі нмэ дѓкмШмЧф

эмШммлммэ ЫмѓбЧхЧ дммЧЭмаЧ№ УнммпммЧбх = ц смѕкмЧШмѕх омммЯ гммѓмЧс ШмЧсУЯбЧф

ц ЪммЭммЧцб ЧсУЯэммЧф эмЧ асЧ № ц умЧ = эмѕЬмЯэ Чсгмнммэмх ЪмѓпмЧЫмѕб ЧслімсмуѓмЧф

гмомѓимЪ ууЧсмэмп ЧсдТу ц ЭѓмцсмѕхЧ = Ухмммс ЧсЬммвэмбЩ ЭЧсмѕхѕму УШмпмЧфмэ

ц ШмсЧЯ улбШімфЧ ЧспммШмэмб ЪмнмѓбјѓомЪ = цЧсммаЦммШ эимсШѕ оЧеіэѓ ЧсЭіумсЧф

ц ШммсЧЯѕ уемб ЪмЯЭбЬмЪ ЭѕпЧухЧ = уммхммвцуммЩё нмэ гЧЭмЩ ЧслммбШмЧф

ц ШмпмэмЪ эЧ эуфэ ЧсгкмэЯ ц сЧуѓфіэ = оммцуё ЪмѕкмЧЪмШммфмэ кмсь УдммЬЧфммэ

ШЧсУуг гмЧбЪ экбШё уф УбжхЧ = ц Чсмэмцу сЧ кмѓбѓШё ц сЧ кмѕмЬмумѓмЧф

пмаШмцЧ кмсмэмпі ц УШммЯѓсмцп омѓгмѓЧцЩ = эмЧ уф дмѓмбѕнмЪі Шмфмгмсмп ЧскмЯфмЧф

Чссх сммц умЬммЯ ЧскммбцШмЩ эмфмЪмдмэ = гммэмнмЧ№ эмомжјѕ умжЧЬк ЧсмбјѕхмШмЧф

ц эммѓкмѕцЯ смсХгмсЧу эммѕгмсмуѕ фммѓнмгмх = ЭмЪмь ЪмѓШміэмфѓ умкмЧЯфѕ ЧсдмјѕЬммкммЧф

ц эЮијѕ нцо ЧсУбж бгу дЬЧкмЩ = ц эммвэммсѕ кммфмхмЧ гЧгмЩ ЧсЮмасЧф

гммЧбЪ Хсь уЯбэмЯ инммсмЩ экмбШ = ЪммШммлмэ вцЧЬ ЧсммЭмо ШммЧсмШмхмЪЧф

ЪмШмлэ Чсгмомци жЭмэЩ ц нбэмгмЩ = ц ШммфмЧЪ Тць гммЧШммомЪ смімумѓпмѓЧф

эммЧ дмцо умЯбэмЯ ЧсмЪэ нмЧбоммЪмѕхмЧ = ц УШмммц ЧсммШмомЧС фмѓЭмімэмњмШмѕх УШмпмЧфэ

омммЯ пмЧф ШмфмэмЧфмЧ Уоммэмму нмуЧ Убь = уммф ЬммЧС эмѓмбЫмѕцЧ ШмкммЯѕх ШмѕфмэмЧфэ

ц Чсфмоеѕ нэ Юмѕсоэ Чсаэ жмэкмЪх = уммЧ Ъмѓму сэ умф ШкмЯх фммоммемЧфммэ

пмму ЯЧбЪ ЧсУэммЧу ЭмЪмь УЭмпмумЪ = ЭѕмвфмЧ ЪмоммсмШ иммбнмхмЧ УвумЧфммэ

ХаЧ умсмцп ЧскмбШ гмЧнмбЩё ц омЯ = ШмммЧкмммЪ Шммммас кммвЩ ЧсмЪмэмЬммЧф

ЬмЧСЪ Ъммсмца ШмбЧхШ уф ШкмЯ умЧ = аЧоммЪ уммбЧбЧ№ умф Шммфэ гммЧгммЧф

эЧ Ухммс омбиШЩ Чсмммаэмф УЭммШммхмму = Чсэмммцу эмѓнммЪммѓЭ ЬмѕбЭмѓпму УЮмцЧфэ

ц бЫмммЧСѕ бфммЯЩѓ дмЧхмммЯё сЧ эфЬмсмэ = пЧсШммЭммммб сэммг ЪЭмЯх дммимТфммэ

цугмЧЬммЯё кмујѓ ЧсУгь ЬмѕмЯбЧфмѓмхммЧ = Ъмдмпмц цЪШмпэ гиммцЩ ЧсејѕмсШЧф

омЯ пЧф нэ ЧсЭубЧС эгпмф УЮмцЪэ = цЧсэмцу еЧб ШомШмжмЩ ЧсгмЬмЧфмэ

оммЯ пмЧф еШЭхму ц УбкЯ Эмцсхму = нмэ Ьмцні гммЧШмЭмЩђ ШмсЧ ЮімсЬмммЧф

цбУэмЪ дЧиШмЩѓ Чсксмцу Ъмсмцумфэ = ц ЧсдммЧиШмммэ ШеммуЪммх УЯумЧфмммэ
