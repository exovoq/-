εα ΫάΗΟΡ ΗαΤεΟΗΑ γάδ γΚάΡΟγ = Γγ εάα ΪΡέάΚ ΝήνήάΙ ΗαγΚίάαάγ
ΣδάΙ Ϊαάμ ΣδάΙ ΚΡΗίάγ έζήεάΗ = ΚΪΘ ΗαΨΡνή ζΣάζΑ ΝάΗα ΗαγΣαάγ
ήγγ ΚυΤσνψσάΟυ έάζή ΓΡΦ ΞΦζΪδάΗ = ΓΡΓνΚ ήΥάΡΗπ νυΘΚδάμ έάν ήγήάγΏ!
νάΗ ΟΗΡ γΓΣάΗΙ ΗαΤΪάζΘ Κίαγάν = ζ Ϊγν ΥΘΗΝ ΗαΠα ένδάΗ ζΗΣαγάν
γάΗ Θνάδ γΔΚγάΡ ζγΔΚγάΡ δάΡμ = ΤΘΝάΗπ νΪΘάΡ Ϊάδ ΞνάΗα γΘεάγ
ΤΜάΘ ζ ΕδίάΗΡ ζ Καάί ΝίάΗνάΙ = γΗΚάΚ αΚΝνάΗ ΥΡΞάΙ ΗαγΣΚΣαάγ
ΓΓΘΗ ΗαέζΗΡΣ ζΜάε ΪΘαάΙ ΤΗΝάΘ = ζ ΓγάΗγ ΞνγΚεάΗ ΝΘΗΖάα γάΜάΡγ
ΚΡδζ Εανάί ΗαΞνάα ζεάν ΝΘνΣάΙ = ΚΤίάζ Εανάί ΘΪάΘάΡΙ ζΚΝγάΝάγ
εαΗψ ΫΣαΚ ΗαΣνέ γδ ΥάΟΓ ΗαΛάΡμ = ζΪέΚ έν ΗαγνΟΗδ ΡίάΦ ΗαΓΟεάγ
εάαΗψ ΓΛάΡΚ ΗαδήάΪ ΝΚάμ νδΜαάν = Ϊάδ ήΘάΝ ζΜάε ΗαΞΗΖάδ ΗαγΚαΛάγ
ΓΓΘΗ ΗαέζΗΡΣ ΓγΨάΡΚ γάδ ΘΪΟίάγ = ΣΝΘ ΗαεΟμ ΫνΛΗπ εδάνΑ ΗαγζΣάγ
αζ ΓΘΥάΡΚ ΪνδάΗί ζΜάε γΝγάΟ = ζ ΡΓνΚ γάΗ νΜάΡν ΘάΟΗΡ ΗαΓΡήάγ
ζΡΓνΚ γίΙ ζεάν ΚΫΣάα ζΜεεάΗ = ΘΗαδάζΡ γάδ ΒΛάΗΡ ανάα γΩάαάγ
ζέΚΝάΚ δΗέάΠΙ αΚΣγάΪ γάΗ ΚάαΗ = ΜΘΡνάα γάδ Βν ΗαίΚάΗΘ ΗαγΝίάγ
ζΡΓνάΚ ίνάέ ΫάΟΗ ΘάαΗα ΣνάΟΗ = πζγΦμ ΗαΨΫΗΙ Εαάμ ΤένάΡ Μεδάγ
αάζ Γδ Ϊνδάί ΓΘΥάΡΚ ΕΣαΗγδάΗ = αΞΡΜΚ γδ ίεέ ΗαΦάαΗα ΗαγΪΚάγ
ζΝγαΚ ΪΘαάΙ ζ ΗαΝΜάΗΘ ννΟεάΗ = ΤΡέΗπ ζΓΨέάΓΚ ΗααΩάμ έάν γάγ
ΓΓΘΗ ΗαέάζΗΡΣ ήάΟ ΪΡέΚάί ΝΗέΩάΗπ = Νή ΗαΜζΗΡ ΚΫάΦ ΨάΡέ ΗαΓίάΡγ
ΓΓΘάΗ ΗαέάζΗΡΣ ήάέ γίΗδάί ΕδδάΗ = αδΪνΤ έάν γάδ ΗαΞάΟΗΪ ΗαγΘάΡγ
αάγ νάΟΡί ΗαΪΡΘάν έάν ΓνΗγάδάΗ = ίάΡγ ΗαΜάΟζΟ ζαΗ νήνάδ ΗαγΣαάγ
ΨυΪφδσάΚ ίΡΗγάΙ ΓγΚάν έάν ήαΘεάΗ = ανΣ ΗαίΡνάγ Ϊαάμ ΗαήδάΗ ΘγΝάΡψγ
νΗ ΓγάΙ ΗαΕΣάαΗγ εάα αάί έάΗΡΣ = νΫΤμ ΗαζΫμ ζνΪάέ ΪδάΟ ΗαγΫδάγ
Εδάν ΠίΡΚάίφ ζ ΗαΜάΡΗΝ δάζΗεάα = γδν ζΝΡέν ήΟ ΚαΜαάΜ έάν έγάν
ΗαήγάΙ ΗαίΘάΡμ ‘ ΜεάΗΟρ ΥάάΗΟή = ζΘδάΗΑ ΥάΡΝ ΕΞΗΖδάΗ ΗαγΚάεάΟγ
ΓγάΗ γΨάΗΡΟΙ ΗαάΣάΡΗΘ έΕδάεάΗ = ζεάγ νΜΡΪδάΗ ίάΔζΣ ΗαΪαάήάγ
γάΟζΗ Εαάμ ΗαΡΝγάδ ΓνΟνίάγ έγάΗ = ΞΗΘάΚ νάΟ ΚγΚάΟ δΝάζ ΗαγδάΪάγ
