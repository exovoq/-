ЧсЬвС ЧсЫЧсЫ уф гсгсЩ ЧсгэбЩ ЧсфШцэх

(инцсЪх есь Чссх ксэх цгсу)
Хфэ ШуЯЭ ЧсУпмммбуэммф уЪэммммммуѕ == цуЭуЯѕ ѕ нэ ЧскЧсуэф ЧсУпмммммммбуѕ
нЭгШхѕ еммммммсь ксэммммх Хсммммхх== цУфх нэ Чсубгсэф ЧсЮммммммммммммЧЪуѕ
Чссх Ькс ЧсиммЧхбЧЪ Эумммсммммфх == цЬЯцЯѕх УихмммммммммЧб ЭЪмммь ТЯуѕ
нХаЧ гспЪ ибэо уммммЯЭ уЭумЯђ == нЧксу ШУфп нэ Чсибэо ЧсУоммммцуѕ
УЪь ЧсфШэ Хсь ЧсЭммэЧЩ смммммэбкмхѕ== бШ ЧсцЬцЯ ноЯ УЪммммммммЧхЧ уэЪмуѕ
ЬЧСЪ Ьуцк ЧсубжкЧЪ сумпмЩђ== эУЮаф уф ху нэ ЧсбжЧк Хсэхѕуѕ
сэпцфцЧ неЭЧС ЧссгЧф УемЭммЩ== нЧспс уф неЭь ЧсШцЧЯі ЪксмммммуцЧ
нШЭЫф нэ ЧсУцсЧЯ кф аэ цЧсмммЯђ == Хф цЯјхѕ уф Юэмммбх оЯ эфкмммммуцЧ
ХсЧ ЭсэумммммЩ УдноЪ ШуммЭммммммуЯђ == цхц ЧсэЪэмммму Шлммммммэб УШђ элбуѕ
нбжЪ Шх цШЭЧсх цЪцпммммммммсЪ == нЬвЧхЧ бШѕј ЧскЧсуэф Чсммммммуфкуѕ
цЧсЮэб ку ј ЯэЧб гкЯ кфммммммЯуЧ == бЬкЪ ЭсэуЩ ѕѕЯЧбхЧ цЧсоммммЧгуѕ
цдЧЪхЧ уф ШкЯ жкн УеШмЭЪ == Ъгоэ Чселэб ук ЧспШэб цЪикуѕ
фвсЪ усЧЦпЩ ЧсХсммммммммх ШУумммбх== сэихбцЧ омммммсШ ЧсфШэ нэкммеуѕ
нУугпмммммммммЧ еЯб ЧслсЧу цдохѕ == ц УЮбЬЧ уф осШх пммммммммЧсуйсуѕ
ЮЧнЪ ЭсэуЩ ѓ Уф эуцЪ уЭумммЯѕ ѕ == ноббЪ ЪѕбЬкх упЩ ѓ гмммммммЧсуѕ
ноЯ ЧсфШэѕ цхц ЧШф гЪђ Уумммммх == нмммммммаЧо уф убіј ЧсвуЧф ЧсуФсуѕ
бШЧх нэ кЧуэф кШЯ ЧсуимммммсШ== нпмммммЧф ЬѓЯѕ ѕ цпЧф гэЯѕ хЧдмммуѕ
бЧЭ ЧсфШэ ЧсдЧу бноЩ кмммммммух == нбТх бЧхШѓ нэ Чсибэо укмммммммсуѕ
суЧ бУь ЧсУдЬЧб уЯЪ йсхммммЧ == цгЭЧШЩ нцо ЧсмммммммммммммлсЧу Ълэуѕ
УЪь ШЭэбь Хсь ЧслсЧу цгУсммммх == кф ЭммммммммммммЧсх угЪэойѕ ѕ Уц фЧЦуѕ
бЯѓѓ ЧсфШэ ШуЧ эиЧШо ценмммммммх == цШЭэбь нэ ксу ЧсфеЧбь кЧсуѕ
пдн ЧсбЯЧС кф ЧслсЧу нцЬмЯх == нэ йхбх УЫб ЧсфШцЩ ЮммммммммммммЧЪуѕ
УбЬк Шх сЯэЧб упЩ гмммммммммЧсуЧ№ == ндУфх нэ ЧсфЧг дммммммммУфѕ ѕ Укйуѕ
УбЬк нсц ксу ЧсэхцЯ ШУуммммммбх == оЯ эупбцЧ ШЧсЭо гЧкЩ экммммсмуцЧ
Чссх кхЯ Хсь Чсбгмммцс ШЭмнйх== оЯ кЧд Шэф ЧсйЧсуэф цоЯ кммуцЧ
