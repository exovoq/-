уЭЧцсЩ ссХкбЧШ
ХаЧ : УЯЧЩ дби лэб ЬЧвуЩ цхэ йбн ссвуЧф ЧсугЪоШс
Уѕснѓ: нкс уЧж уШфэ ссуЬхцс уШфэ ксь ЧснЪЭ ( нкс Чсдби )
ЧсдЦѕ: фЧЦШ нЧкс убнцк ШЧсжуЩ ЧсйЧхбЩ
ЧгЪхЧф: нкс уЧж уШфэ ксь ЧснЪЭ ( ЬцЧШ Чсдби )
Шх: ЬЧб цуЬбцб уЪксоЧф Ш ( ЧгЪхЧф )
ЧснЪь : нЧкс убнцк ШЧсжуЩ ЧсуоЯбЩ ксь ЧсУсн
нсѓу : ЧснЧС гШШэЩ Уц кЧинЩ Ё су: УЯЧЩ фнэ цЬву цосШ.
эбхѕ : эбѓ нкс ужЧбк уЬвцу ШкЯ су ШЭан Эбн ЧсксЩ цЧсхЧС жуэб уЪес уШфэ ксь Чсжу нэ уЭс феШ ункцс Шх.
ШФгь : ункцс Шх уоЯу сснкс ( эѕкѓЯјѕ ) уфецШ ШЧснЪЭЩ ЧсуоЯбЩ ксь ЧсУсн.
экЯјѕ: нкс ужЧбк уШфэ ссуЬхцс убнцк ШЧсжуЩ ЧсйЧхбЩ цфЧЦШ ЧснЧкс жуэб угЪЪб ЪоЯэбх ( хц ) цЧсЬусЩ нэ уЭс феШ ункцс ЫЧф сснкс эбь.
цсЧ фкуь: ЧсцЧц кЧинЩЁ сЧ фЧнэЩЁ фкуь укицнЩ ксь ШФгь уфецШЩ ШнЪЭЩ уоЯбЩ ксь ЧсУсн.
