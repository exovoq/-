{ وَلَا تُكْرِهُوا فَتَيَاتِكُمْ عَلَى الْبِغَاء إِنْ أَرَدْنَ تَحَصُّنًا لِّتَبْتَغُوا عَرَضَ الْحَيَاةِ الدُّنْيَا وَمَن يُكْرِههُّنَّ فَإِنَّ اللَّهَ مِن بَعْدِ إِكْرَاهِهِنَّ غَفُورٌ رَّحِيمٌ } النور/33

{ وَلَا تُكْرِهُوا فَتَيَاتِكُمْ عَلَى الْبِغَاء }

المقصود بالبغاء الزنا, فالبِغاء مَصْدَرُ باغت بِغاء إذا زنت.

في القاموس :

" وبَغَتِ الأَمَةُ تَبْغِي بَغْيًا وباغَتْ مُبَاغاةً وبِغاءً فهي بغِيٌّ وبَغُوٌّ : عَهَرَتْ . "


إضافة :


{ قَالَتْ أَنَّى يَكُونُ لِي غُلَامٌ وَلَمْ يَمْسَسْنِي بَشَرٌ وَلَمْ أَكُ بَغِيًّا } مريم/20

{ وَلَمْ أَكُ بَغِيًّا }

تجمع الكلمة ( بغي ) على بِغاء, و بغايا أيضًا.


الخلاصة :

الكلمة ( بِغاء ) يجوز أن تكون مصدرا للفعل باغت, ويجوز أن تكون جمعا للكلمة بغي.
