ΓΚΚδάν νάΗ ΓΝΘάΙ έάν ΡΔΗεάάΗ = γάδ ΗαΒαΗγ γάΗ Γζεάμ ήάζΗεάΗ
Ϊαμ ΜΣΡ ΗαΤΜζδ αεΗ Γδνδ = ΓΪααεΗ έάαάάγ Κάγάεάα ΓΞάΗεάάΗ
ΓΚΚδάν ΚΞέάή ΗαΒεάάΗΚ ένάεάΗ = ΚζΤζΤδν έαγ ΓΪΡέ γδΗεάΗ
ΚΜΗέΚ ζΗΡΚγΚ ίγΟΗ ζέΡΗ = Ϊαμ ΓΪΨΗέεάΗ ΚΘίάν ΪαΗεάΗ
ΚΟζΡ ζέάν γΝνΗεάΗ ΗδίΣάΗΡ = νΨάζψή ΜνΟεάΗ ΝνδάΗ ζέάΗεάΗ
ΘαΝΩ ΗαΤΡ ΚΡγήδν ζΚέΡν = ίνΗδάν ίαγάΗ ΚΫάΤάμ ΡΔΗεάάΗ
γδ ΗαΓΪγάΗή ΚΘΪΛεάΗ δΤνΜάΗ = ζέν ΗαΒέάΗή γδίΣάέ ΣδΗεάΗ
ΓΚΚδάν ήαΘάεάΗ Ξάέάή ζΝάάδ = ζΥνΝάΗΚ ΚάΡΟΟ έάν γΟΗεάΗ
Ϊαμ ζΪΟ ΗαΣΡΗΘ αεΗ γΣνΡ = ζγζΜ ΗαίΡΘ ΘΗαΪΚΘμ ζΗεΗ
ΓΚΚδάν ζΗαήνάζΟ Ϊάαάμ νΟνάεάΗ = ΚΚΡΜάγ γάΗ ΚδάζΑ Θάάε νάΟΗεάΗ
έΜΗΤΚ Ννδ ΟΫΟΫεάΗ δΣνάγ = ζΠίψΡεάΗ ΘΨνάΘ γάδ ΤΠΗεάΗ
ΓΚΚδάν γάδ ΚΜΗΪνάΟ ΗαένάΗέάν = γΣεάΟΙ ΚυΣΡΘάα έάν ΞΨΗεάΗ
ΚΞάΨάΚ ααγάΒήάν ζΗΣΚάήάΡΚ = Ϊαάμ ίΘάΟ ΗαγΪδψάμ έΗΝΚζΗεάΗ
ΓΚΚδάν ζΗαΟγάζΪ αεάΗ ΝάάΡΗί = Ϊαάμ ζΜδΗΚεάΗ ΓΘάαάμ ΘεάΗεάΗ
ζΣΗΡΚ Θνδ ΓγάζΗΜ ΗαΚδάΗΟν = έΚΗεάΚ αΗΚΪάν ένεάΗ ΥΟΗεάΗ
ΓΚΚδάν έάν γΫΗδνεάΗ δάΦάζΘ = ζΡνΝ ΪΗΥέ ΪάΡψμ δζΗεάΗ
εδάΗί ΚγαγαάΚ ΚΜΚάΡψυ ΫνΩάΗ = ζΥΘψΚ έν ΚΜΗζνάέ ΣήΗεάΗ
ΓΚΝΣάζε Ϊαάμ ίάΟΡ ΗαανάΗαάν = ζΚΡΣαάε Ϊάαάμ ΟΗΑ ΘάΡΗεάΗΏ
ΓΚΚδάν νάΗ ΓΝΘάΙ ήάάΟ ΚάΟΗΪάΚ = γΪΗδνάεάΗ νΥάζψΡεάΗ ΜάζΗεάΗ
ζΥΗΝΚ γαΑ ΓέζΗε ΗαΘζΗίν = ΓΚΝΡγ έν ΗαΪζΗαγ γΤΚεΗεάΗΏ
εν ΗαΝΗα ΗαγΡνΡΙ δΝΚΣνεάΗ = ΜΡΚ έν ΓγΙ ΦΪέΚ ήζΗεάΗ
