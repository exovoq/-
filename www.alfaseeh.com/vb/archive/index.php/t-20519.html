ГнЬНЬбЬЬж ЪЬдЬПЯЬЬг ЫуЬЬЬЬТубхї!!
ГнЬНЬбЬЬж ЪЬдЬЬПЯЬЬг ЫЬЬЬЬЬТбх = ЭЬЮЬбЬИЬн ЗбЬЬнЬЬЬжг гЬФЬКЬЪЬЬбх
КЬЬжЭшцЬЬС гЬеЬМЬКЬЬн дЬЬЬЬЬЗСс = бнЬНЬСЮЬеЬЗ ЗбЬЬеЬЬжм ЗбЬЪуЬМцЬЬбх
ЭЬЬгЬЬд нЬШЬЭЬЬн НЬЬСЗСКЬЬЬеї = ЭЬЬгЬЬд бЬЬбЬЬСжН нЬдЬКЬФЬЬбхї
ЪЬЬРЗИ ЗбЬЬЬСжН гЬЬЬЗ ГЮЬУЬЬм! = нЬЬгЬЬТшЮ ЯЬдЬеЬеЬЗ ЗбЬОуЬЬШуЬЬбх
КЬЬЬРжИ ЗбЬдЬЭЬУ гЬЬЬд НЬЬЬСЮ = ЭЬнЬИЬЯЬн НЬТдЬеЬЬЗ ЗбГМЬЬЬЬЬбхх
жКЬЬдЬЬПИ НЬЩЬЬеЬЬЗ НЬЮЬЬИЬЬЗ = ЭЬЯЬнЬЬЭ нЬЬгЬЬжК РЗ ЗбГгЬЬЬЬбх
ЭЬТеЬСКЬЬеЬЬЗ РжК ЪЬЬШЬЬФЬЬЗр = КЬЫЬнЬЬЛ ЗбЬЬгЬЬЬЗБ нЬСКЬЬНЬЬбх
жКЬУЬЯЬЬд ВеЬЬЬЬЬЙ КЬЬНЬЬЬПж = КЬЪЬнЬЬП ЗбЬЬНЬЬЬТд нЬдЬеЬгЬЬбх
СНЬЬнЬЬЮ ЗбЬЬНЬЬИшц КЬУЬЭЬНЬЬе = ИЬЬПгЬЬЬЪ гЬЬЗбЬЬЬе НцЬЬнуЬЬЬбх
нЬЬОЬЬЬПшцП ОЬЬЬЬЬПш ЪЬЗФЬЬЮЬЬе = ЭЬКхЬдъЬНцЬбх МЬУЬгуЬЬе ЗбЬЪЬЬбЬЬбх
ИЬФЬЬЗФЬЬКЬЬе гЬЬХЬЬЬЭшЬЬЬПЙс = ИЬРЯЬЬСм гуЬЬЬдъ ИЬеЬЬЗ СНЬбЬЬжЗ
ЭЬнЬгЬСЦ ХЬИЬСеЬЬЗ КЬЬХЬЬПм = ЭЬнЬХЬЬСО ФЬжЮхЬеЬЬЗ ЗбЬОуЬИцЬЬбх
ЪЬЬЬСК ГНЬЬбЗгуЬЬе ЫхЬЬХуЬЬХс = ГнЬЬЗ ХЬИЬЬСн ЭЬгЬЬЗ ЗбЬЪЬгЬбх ї
ЭЬЬеЬЬРЗ ЗбЬЮЬЬбЬЬИ гЬУЬЬЯЬЬжд = ИЬЬНЬЬИ ЗбЬдшхЬЬМъЬЬбц гхЬЪъЬКуЬЮЬЬбх
ЭЬЬЬЬЬбЗ ЪЬЬЬЬЬРбс бЬФЬеЬЮЬКЬе = жбЗ ИЬгЬЬгЬЬЗКЬЬе ЮцЬЬЬЬИуЬЬЬЬбх
КЬЮЬбшцЬЬИЬЬЬЬе гЬЬЬЬСЗСКЬЬЬЬе =жКхЬУъЬецЬЬС ЪЬнЬЬдЬЬе ЗбЬЫцЬЬнцуЬЬбх
ФЬгЬЗКЬЬЙ ИЬЪЬЬЦЬЬе ЭЬЬнЬЬЬе = КЬЬгЬЬМшх ПгЬЬЬЬЬЗБе ЗбЬУЬЬИЬЬбх
гЬЬФЬЬм ИЬЗбЬОЬШЬжЙ ЗбЬЛЬЯЬбЬм = КуЬЬЖцЬЬдшх ИЬбЬНЬЩЬЬе ЗбЬгхЬЬЮуЬЬбх
ЭЬЪЬдЬЬПн ЮЬЬЗгЬЬЬЙ дЬНЬЬбЬЬК = жЪЬЬдЬЬПн ЗбЬбЬнЬЬб гЬКЬЬХЬЬбх
жЪЬЬдЬЬПн ЗбЬУЬеЬЬП гЬдЬКЬУЬИ = жЪЬЬдЬЬПн ЗбЬЬдЬЬжг нЬИЬКЬеЬЬбх
жЪЬдЬЬПн ЗбЬЬНЬЬСЭ гЬУЬКЬЪЬСс = ИЬЬИЬЬДУ ЗбЬеЬМЬЬС гЬдЬФЬЫЬЬбх
нЬбЬШшЬОЬе ЗбЬЬдЬЬжм жнЬЬЬЬЬСм = ИЬЪЬнЬЬд УЬЗгЬЬеЬЬЗ ЗбЬЛшцЬЬЮуЬЬбх
ИЬЬЯЬЬГУ ЗбЬЬХЬЬП гЬСКЬЬеЬЬд = ИЬЬЪЬЬЬбшт ФЬЬСИЬЬЬе ЛуЬЬгцЬЬЬбх
ИЬбЬНЬЬЩ ЗбЬЬСЖЬЬг гЬЦЬШЬеЬЬП =ИЬЬФЬЬжЯ ЗбЬЬЬЬжСП гЬЯЬКЬНЬЬбх
нЬеЬПеЬЬПе ЗбЬЬеЬЬжм ЪуЬИцЬЬЮЬЬЗр = ИЬЬРЯЬЬСм ФЬЗИЬеЬЬЗ ЗбЬОЬЬМЬЬбх
УЬЬЬСК ЭЬЬЬн МхЬдъЬНЬе ГбЬЬгЬЬЗр = КЬЪЬнЬЬП МЬСжНуЬеЬЬЗ ЗбЬМЬЬгЬЬбх
гЬЬКЬЬм ЮЬбЬЬИЬЬн КЬШЬЬЗСНЬЬе = ЭЬнЬУЬЮЬм ЗбЬЬЪЬЬРИх жЗбЬЪЬУЬЬбх
КЬЬЪЬЬЬжП ЕбЬЬЬЬЬм НЬПнЬЮЬКЬЬе = ЭЬКхЬУъЬЪцЬЬП ЮЬбЬЬИЬЬе ЗбЬЮхЬЬИуЬЬбх
жнЬУЬеЬЬС гЬЬЬЬЪъ ТхеуЬнъЬЬСКЬЬе = ждЬЬЬЬжС ЗбЬЬИЬЬЬПС нЬЯЬКЬгЬЬбх
жнЬЬСУЬЬб ЪЬЬнЬЬЬд бЬнЬбЬКЬЬе = бЬнЬНЬСУЬЬдЬЬЗ жнЬЬдЬЬЬУЬЬЬПбх
ЯЬЬГдшЬЬЬЗ УЬЬЬЬЬС ЩхЬбъЬгуЬКцЬЬе =ИЬЬдЬЬЗ ЗбГФЬЬЬЬжЗЮ КЬФЬКЬгЬЬбх
жФЬЬЪЬЬСн ИЬЬЬЬЗК гЬСКЬУЬгЬЗ =ИЬИЬНЬЬС ЗбЬЬжХЬЬб нЬЫЬКЬУЬЬбх
нЬЦЬЬнБ ЗбЬЬНЬЬСЭ гИЬКЬеЬМЬЗ = нхЬЬСЮшЬЬХ ЮЬбЬИЬдЬЬЗ ЗбГгЬЬЬЬЬбх
жКЬЬЬТеЬЬЬС жСПЙ ЫхЬЬЬгцЬЬЬСК = ИЬГЪЬдЬЬЗд ЗбЬУЬЬгЬЬЗ ЯхЬЬбуЬЬЬбх
жнЬЬФЬЬСИ ЯЬждхЬдЬЬЗ ЛуЬЬгцЬЬЬбЗр = ЭЬнЬУЬЪЬП НЬИшхЬЬдЬЬЗ ЗбЬЛшуЬЬгцЬЬбх
