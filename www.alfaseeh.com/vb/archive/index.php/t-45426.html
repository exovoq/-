ШнС ЗбдХС

ГШнС ЗбдХС гЗбЯ ЫИЬЬК ЪдЬЗ = жЮЬЬЬП НбЮЬЬК ПеСрЗ ЭЬЬм УгЬЬЬЗдЗ
ГКдУЬЬЬЬЬм нжг Гд ЯдЬЬЗ гбжЯЬЬЗ = дУЬЬЬЬжУ ЗбГСЦ жЗбПднЬЬЗ ТгЗдЗ
ГКдУЬЬЬЬм ОЗбЬЬПЗ УнЬЭЗ КжОЬЬм = СЮЬЬЬЗИ ЗбЯЭЬЬЬС гдеЬЬЬЬг жЗбИдЬЬЗдЗ
ГКдУм ХЬЬжбЙ ЗбЮЬЪЬЮЗЪ ЭнеЬг = ЫЬЬЬПЗЙу ЗбЮЗПУнЬЬЬЬЬЙ еЬЬб КЬЬЬЬЬжЗдЗ
ГКРЯС нжг Гд дЗПК жХЗНЬК = ИгЪКХЬЬЬЬЬгт бЮЬЬЬЬП РЮЬЬК ЗбеЬЬжЗдЗ
ГМЬЗИ беЗ ЗбдЬЬПЗБ ЭПЗе дЭУЬЬм = ИМнЬЬЬЬФт ббЯСЗгЬЬЬЬЬЙ ЮЬЬЬП КЭЬЬЬЬЗдЗ
дОнЬЬбх ЗбгЬМП ндИхК Эм СИЗдЬЬЗ = ЛгЬЬЬЬЗСх ЗбдХС КхСжм гд ПгЬЬЬЗдЗ
дОжЦ ЗбгжК Гж Нгг ЗбгдЗнЗ = СЦЬЬЬЬЗ ЗбСНгЬд Эм ЗбПднЗ СМЗдЗ
ЭдХЬЬСс Эм ЗбНнЗЙ жЕ д ЮКбдЬЬЗ = ЭЬЭЬЬЬЬм МдЬЬЬЬЬЬЗК СИЬЬЬм гбКЮЬЗдЗ
ЭСПш ЗбШЬЬнС НЮЬЬЗ р гЗ РЯСКЬЬг = жбЯЬЬЬЬЬЬд РЗЯ ЪеЬЬЬЬЬПс бЗ нЬЬЬЬПЗдЗ
ЭеЬЬг ббЗгЬЬЙ ЗбЫСЮЬЬм дМЬЬЬЗЙ = жеЬЬЬг ЭЬЬЬм ЗббнЬЬбЙ ЗбЩбгЗ ЦнЗдЗ
ЪТЗЖгЬЬег КМЬЬЗжТК ЗбЛСнЬЬЬЗ = жИЗбЕнгЬЬЬЬЗд ЮЬП ИбЬЬЬЫжЗ ЗбЪЬдЬЬЬЗдЗ
жНИ ЗбПнд Г жСЛег ФгжОЬЬЗ = жИЫЬЬЦ ЗбгЬЬжК ГжСЛЯЬЬг ежЗдЗ
СЦнКг ИЗбОджЪ жгЗ деЦКЬг = дЬЬЬЬежЦ ЗббнЬЬЛ нЬЬГИм Гд неЗдЗ
ЭШЬЬнС ЗбдХС нЪФЮ Яб НЬС = жбЯЬЬЬЬЬд Эм ЗбгЬЬЬЬРбЙ бЗ КЬЬЬЬСЗдЗ
гРЗИЬЬН ИЗбОбнЬЬб жИнК бНЬЬг = гЬЬЬМЬЗТС ЮП ГЮнгЬЬЬК нЬЬЬжг ЮЬЗдЗ
жЫТЙ КУКЫнЛ ИГЪбм ХжК = МдЬЬЬжП ЗбЯЬЬЬЭС КСКЬЬЪ Эм НгЗдЗ
ИГЮПЗг ЗбдМЗУЬЬЙ ПнЬУ КСИЬЬм = КЬПдЬЬЬЬУ Яб ФИЬЬЬЬС гЬЬд ЛЬСЗдЗ
жЯЗИжб ЗбКм Г ЦНК ОСЗИЬЗ = ЕбЬЬЬЬЬм СИ ЗбИСнЬЬЬЙ гФКЯЬЬЬЬЬЗдЗ
жШЭЬб ЗбгУбгнд ЫЬПЗ ФСнПЗ = ШЬЬЬСнПЗ бЗ СЪЬЬЬЬЗнЙ Г ж НдЬЬЗдЗ
Ъбм бЬИд ЗбгЬеЗдЙ ЮП КСИЬЬм = жЪЬЬЬЗФ ЗбгЬЬЬеП РбЗ жЗгКеЬЬЬЗдЗ
ЭнЬЬЗ жнб ЗбШЬЭжбЙ гд ГдЬЗУ = ГЦЬЬЗЪжЗ ЗбТеСЙ ЗбГжбЬм ЪнЬЗдЗ
ГКЬбЯ ШЬЬЭжбЙ ЗбЕУбЗг ЭнЯЬЬг = ЭгЬЬЬЬЬЗ ИЬЬЗб ЗбФИнИЬЬЬЬЙ нЗ ГОЬЬЗдЗ
УгнЙ Эм ЗбФИЗИ КгжК дУнЗ = жКНнЬЬЬЬЗ ЭЬЬЬм ЮЬЬбжИЬЬЬег ПнЬЬЬЗдЗ
ГСгТЗр ббШЬеЗСЙ ЮЬП ГЦЪКЬЬг = жУЬЮКЬЬг ббСРнбЬЬЬЬЬЙ гЬЬФКЬЬЬЬеЗдЗ
ГОЬЬЗ ЗбЗУбЗг гЗ НбЮКх нжгЬЗ = жбЗ СЭСЭК нжгЗр Эм ГМжЗБ ЯЗдЗ
ЭгЬЬЗ ЗбгЪдм бгЗЦт ЮП КжбЬм = жНЗЦЬЬЬСЯг КИбЬЬЬП жЗУКЯЬЬЗ дЗ
ЭЯЬЬг гЬд гУКЫнЬЛт ЭЬм ЗбИЬСЗнЬЬЬЬЗ = ЭбЬЬг нМЬЬП ЗбМЬжЗИ ГжЗбИнЬЬЬЬЬЗдЗ
ЭнЬЬЬЬЬЬЬЗ ббе гЬЬд ХгЬЬЬКт СеЬЬЬнИ = ГХЬЬЬЗИ ЗбгУбгЬЬнд жЮЬП ПеЬЗдЗ
ЭГнЬЬд МНЬЬЬЬЗЭб ЗбФМЪЗдц ц гдЯЬЬЬг = ГОЬЬЬЬЗ ЗбЕУЬЬбЗг бЗ КЯЬд ЗбМИЬЗдЗ
ЭИЗбЕЮЬЬЬПЗг нУЬЬЬеб ЯЬЬб ХЪЬЬЬЬИ = жИЗбЕНМЬЬЗг бЬЬд КШЬЬЬГ ЗбМдЬЬЗдЗ
ХбжЗ ЗбгЗЦм ЗбгМнП ИНУд ЭЪбт = жеМЬЬЬСЯг ЗбЯЬЬЬбЗг Иб ЗббУЬЬЗдЗ
жЮЬЬЬЬб ббФгУ КФЬЬЬЬСЮ гд МПнПт = жЮЬЬЬб ббнб нСНЬЬб Ъд УгЬЬЗдЗ
