Уес Чснкс ( ЧФЮа ) ШхувЪэф Ё ЧсУцсь хувЩ Чсцес ЧсЪэ ЪУЪэ спэ фЪцејс ШхЧ ссфио ШЧсгЧпф Ё ц ЧсЫЧфэЩ ЧсхувЩ ЧсУесэЩ уф Чснкс (УЮа) Ё ц сујЧ ЧЬЪукЪ хувЩ Чсцес ( Ч ) ук ЧсхувЩ ЧсУесэЩ (Ф) ц пЫб ЧгЪкуЧс хаЧ Чснкс ЭѕанЪ ЧсхувЩ ЧсУесэЩ ссЪЮнэн Ё нсу экЯ хфЧп ЭЧЬЩ сШоЧС хувЩ Чсцес неЧб Чснкс ( Юѕањ ) . ц хаЧ ЧсЭан гуЧкэ сЧ эфиШо ксь пс ЧсУнкЧс .

ц ЪсЧЭй Уф ЧсжујЩ ксь ЧсЮЧС Ъдэб Хсь Уф Чснкс ужуцу Чскэф нэ ЧсужЧбк ЭэЫ эоЧс : ( УЮѓа Ё эУЮѕа Ё Юѕа ) .
