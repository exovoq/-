гФЧсэ кф ЧспсуЧЪ Уц ЧсУгуЧС ЧсЪэ эупф ХгЪЮбЧЬхЧ уф Чснкс, Уц Уес Чснкс, уЫс:

Чснкс "ЬЧв"

ЧсЬаб = ЬЧв
ЧсуЧжэ = ЬЧв
ЧсужЧбк = эЬэв
ЧсУуб = УЬв
Хгу нЧкс = ЬЧЦв (цЬукхЧ ЬцЧЦв ц ЧсуФфЫ ЬЧЦвЩ ц ЬукхЧ ЬЧЦвЧЪ)
Хгу ункцс = ПППП
ПППП = ЬцЧв ( Уу УфхЧ уф Чснкс ЬвьП)
ПППП = УЬэв (жуЩ ксь ЧсУсн)
ПППП = УЬЧв (уЧж, нЪЭЩ ксь ЧсУсн)
ПППП = ХЬЧвЩ
ПППП = эЬцв ( хс "эЬцв" УЪЪ уф Чснкс "ЬЧв" УесЧП )
