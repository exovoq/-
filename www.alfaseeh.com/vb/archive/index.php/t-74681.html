ксс Чсуфк уф Чсебн
УцсЧ: ЧсУксЧу
1- ЧсксуэЩ цЧсЫУфэЫ (ЮЯэЬЩ- укЧцэЩ - гкЧЯ - хфЯ )
2- ЧсксуэЩ цЧскЬуЩ (бугэг - ХШбЧхэу - эцгн)
3- ЧсксуэЩцЧсУсн цЧсфцф ЧсвЧЦЯЪэф (кЫуЧф - дкШЧф )
4- ЧсксуэЩ ксь цвф Чснкс (эвэЯ - УЭуЯ Упбу)
5- ЧсксуэЩ ксь цвф (ніі№№ђкс)(куб- ЬЭЧ-вЭс)
6- Чсксу ЧсубпШ ЪбпэШЧ увЬэЧ (ШксШп - ЭжбуцЪ - ШцбгкэЯ)
ЫЧфэЧ:ЧсенЧЪ
1- енЩ ксь цвф (нксЧф )цуФфЫЩ(нксь)(иуТф - ЬцкЧф)
2- енЩ ксь цвф (Ункс)ЧсЪнжэс (УпШб - Унжс )
3- енЩ ксь цвф(Ункс ) цуФфЫхЧ (нксЧС)(УЭуб - УЮжб)
4- енЩ уф ЧскЯЯксь цвф (нкЧс ц ункс )(уЫфь - ЫсЧЫ - бШЧк)
ЫЧсЫЧ : ЧсУгуЧС
1- ЧсуЮЪцу ШУсн ЧсЪУфэЫ ЧсууЯцЯЩ (еЭбЧС - гэфЧ)
2- ЧсуЮЪцу ШУсн ЧсЪУфэЫ ЧсуоецбЩ ( елбь - пШбь )
3- пс Ьук Ъпгэб Шх Усн вЧЦЯЩ ШкЯхЧ ЭбнЧф (угЧЬЯ - укЧхЯ)
4- пс Ьук Ъпгэб Шх Усн вЧЦЯЩ ШкЯхЧ ЫсЧЫЩ УЭбн УцгихЧ гЧпф (уеЧШэЭ - уэЧЯэф - унЧЪэЭ)
