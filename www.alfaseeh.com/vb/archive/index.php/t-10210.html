Чкбж ЧсфгЮЩ ЧспЧусЩ : Чсункцс укх

цгэЪу ЪцжэЭх Ъксэуэ№Ч цХкбЧШэ№Ч цно ЧскфЧеб ЧсТЪэЩ:
1 м Ъкбэнх
Чсункцс укх хц Чгу нжсЩ уфецШ ШкЯ " цЧц " Шукфь "ук" угШцоЩ ШЬусЩ ЪЭЪцэ ксь кЧус ЧсфеШ нэхК уЫс [ гхбЪ цобЧСЩ ЧсобТф ] обЧСЩ: ункцс укх уфецШ цксЧуЩ феШх ЧснЪЭЩ ЧсйЧхбЩ.
2 м кцЧус феШх
ЪЪкЯЯ кцЧус феШ Чсункцс укх ксь ЧсфЭц ЧсЪЧсэ :
У м Чснкс К уЫс [ гбЪ цУапЧб ЧсеШЧЭ ] УапЧб: ункцс укх уфецШ цксЧуЩ феШх ЧснЪЭЩ ЧсйЧхбЩ.
Ш м дШх Чснкс: цдШх Чснкс хц уЧ эфеШ Чсункцс уф ЧсудЪоЧЪ ЧскЧусЩ кус Чснкс, цхэ:1м Чгу ЧснЧксК уЫс [ УфЧ кЧЦд цхуцу Чсугсуэф] хуцу: ункцс укх уфецШ Шм ( кЧЦд ) цксЧуЩ феШх ЧснЪЭЩ. 2м Чгу ЧсункцсК уЫс [ ЧсдЧкб упЧнУ цЧсоеэЯЩ ] ЧсоеэЯЩ: ункцс укх уфецШ Шм ( упЧнУ ) цксЧуЩ феШх ЧснЪЭЩ.3м ЧсуеЯбК уЫс [ эгбфэ кусп циЧкЩ Чссх ЪкЧсь ] иЧкЩ: ункцс укх уфецШ Шм (кусп) цксЧуЩ феШх ЧснЪЭЩ.4м Чгу ЧснксК уЫс [ бцэЯп цЧсгнэх ] Чсгнэх: ункцс укх уфецШ Шм ( бцэЯ ) цксЧуЩ феШх ЧснЪЭЩ, цЧсукфь ( Уухс фнгп ук Чсгнэх цсЧ ЪЬЧбэх ).
Ьм м "уЧ м пэн" ЧсЧгЪнхЧуэЪЧфК уЫс [ ( уЧ УфЪ цхцЧф ЧсугсуэфП ), ц ( цпэн УфЪ цХоЧуЩ ЧсуЬЪук ксь ЧсецЧШП)] хцЧф м ХоЧуЩ : ункцс укх уфецШ Шм (уЧ м пэн) цксЧуЩ ЧсфеШ ЧснЪЭЩ ЧсйЧхбЩ, цЧсЪоЯэб [(уЧ Ъпцф цхцЧф Чсугсуэф ), ц( пэн Ъпцф цХоЧуЩ ЧсуЬЪук ксь ЧсецЧШ)]


3 м пэн фЭЯЯ Чсункцс укхП
уЪь фкбн Уф ЧсЧгу ЧсуфецШ ШкЯ ЧсцЧц ункцсЧ укх, цсэг ункцсЧ№ Шх Уц укицн№Ч ксь уЧ оШс ЧсцЧцП
ЧсЧгу ЧсцЧок ШкЯ ЧсцЧц сх ЭЧсЧЪ, эхуфЧ уфхЧ ЧсЭЧсЩ ЧсЪэ эЪцЬШ ХкбЧШ ЧсЧгу ункцсЧ укх, цуф УбЧЯ ЧсЧгЪэнЧС нксэх ШпЪШ ЧсфЭц ЧсфйбэЩ. цхах ЧсЭЧсЩ эЪцЬШ феШ ЧсЧгу ксь Уфх ункцс укх сгШШђК уЫс кЯу ЧгЪоЧуЩ ЧсукфьК уЫс [ ЬсгЪ цЭЬбЪэ ] ЭЬбЩ: ункцс укх уфецШ Шм( Ьсг) цксЧуЩ феШх ЧснЪЭЩ ЧсйЧхбЩ.
