αγδ ΗαΪνΟ Ηανζγ Ώ


ΜΗΑΚ ΜζΗΘΗπ Ϊαμ ΣΔΗα ΓΨαήε ΗαΤΗΪΡ γΝγζΟ ΗαδΜΗΡ ΥΘνΝΙ νζγ ΪνΟ ς ήΗΖαΗπ :


( αγδ ΗαΪνΟ Ηανζγ Ώ ) έίΗδΚ εΠε ΗαήΥνΟΙ γδ ΗαΤΗΪΡ


αγδ ΗαΪνΟ Ηανζγ Ώ
(γΝγζΟυ) εΠν ΘΝάζΡ ΗαΤΪάΡ ΚΣΚεάγυ = ζΗαίάζδυ γάδ ΝζαεάΗ ΘΗαεάγψ γΝΚάΟγ


ζΗαΪνΟ ζΗέάμ ΓνάΗ ( γΝγάζΟ ) γαΚΝέάΗπ = ΪΘάΗΑΙπ ΨάΡΗεάΗ ΗαάζΜάΟ ζΗαΓαάάγυ


ζήγάΚσ ΚΣάΓαυ νάΗ ΓΣάΚάΗΠυ γγΚΝάδάΗπ = (αγδ ΓΚμ ΗαΪνάΟ ) ζΗαδνάΡΗδ ΚΦΨάΡγυΏ


ζίάΟΚυ γάδ ΦΜψάΙ ΗαΓΥάΟΗΑ ΛΗΖάΡΙπ = ΓήΦάν ΅ζνΫΡήδάν έάν αυΜψάε ΗαΥγάγυ


ζΗαγαεγάΗΚ ζέάΟδ Ηανάζγ έάν γάάΟΟς = έΗαΞΨάΘ ΓίΘάΡυ γγάΗ νΨαάΘ Ηαήάαάγ


γΣΚΓδΣάΗΚ αάΟΗΡς γάΗ ΘάεάΗ ΓΝάάΟυρ = ζαΗ γάΚάΗΪρ αάδάΗ ένάεάΗ ζαΗ ΡΝάάγυ


ΗαΘάΗΘ ΓζΥάΟ ζΗαΓΝάΡΗΡ ήάΟ ΩΪδάζΗ = ζΤΗεάΟΗν ΘεάΗ ΗαΓΨάάαΗα ζΗαάΡγάγυ


ΓΡμ γάαΗΪάΘ ΓΨάέάάΗας ΚυΝάΟψΛυάδάΗ = ΪδεΗ ΘήΗνΗ Ογάμπ έάν ΣΗΝάΙς ΝΨσγάζΗ


ΓΡμ εδάΗ ΝάγάΙπ ΘάΗΚάΚ γΘΪάΛάΡΙπ = ζΗαέΓΣυ νΡΛάν αΝΘάας έζήεάΗ ΜΠγάζΗ


ζΨΗΖάΡνάδ ΓήάΗγάΗ Λσάάγψσ ζΗΡΚάΝάαΗ = ζΗΝΚάα ΫΥδεγάΗ ΗαΫΡΘάΗδ ζΗαάΡψΞάγυ


Καάί ΗαήάαΗΪ ΗαΚάν ίΗδάΚ γΝΥψάδάΙπ = ΓΦΝνάδ νΝίγδεάΗ ΗαήνδάΗΚ ζΗαάΞάΟγυ


ζΥάΗΡ νΡΨάδ ένεάΗ ίάαψυ Πν αάΫάΙς = ΤζεάΗΑσ ΣζψήεάΗ έάν ΗαήαΪάΙφ ΗαΪΜάγυ


( ΪίΗΩ ) ζ ( ΗαγΡΘΟ ) ΗΣΚΗέΗ ΡγΗα Φδμπ = ζΓΥΘΝάΗ γΡΘΦάΗπ ΚάΓζν Θάε ΗαΫάδάγυ


(ΡΤνΟδΗ) ΘάΗΚ νΝΣάζ γάδ ΣέΗεΚάε = ίάΓΣάΗπ νγΗΜάεάΗ ΗαΕΠαΗα ζΗαάδάάΟγυ


ΞάΡΗΜ Καάί ΗαΫγΗγάΗΚ ΗαΚάν ΠεΘάΚ = γάΗ ΜΗΑδάΗ γδάε αΗ ήγάΝρ ζαΗ Ϊάδάγυ


ΓΦΝμ ΗαΞΡΗΜ Εαάμ ΗαΫΗνάδ ζΜεΚυάευ = ζέάν ΗαΞΥζγάΗΚ ααΪΗΟνάδ δΝΚάίάγυ


(ΥαΗΝδΗ)αάγ νΪάΟ ααάΟνάδ ΫΦΘάΚυάευ = ζΗΣΚΘΟαΚ έν ΗαΝγμ ΗαΓΪάΡΗέ ζΗαδψΩάγυ


γνΟΗδάε ΘΗαεάζμ ΦάΜψάΚ ΜζΗδάΘάευ = ζΞναυάε νγΚΨάν ΥεϊζΗΚάεάΗ Ηαάήάγυ


ζΘάΗΪ ΓήάΟΣ ΓζΨάΗδάν ζΥάΗΟΡεάΗ = ζΜδάΟυευ ίάαψ γΪγάζΡς αάδάΗ εάΟγάζΗ


( ΥαΗΝδάΗ ζΡΤνάΟρ ) ΓέΣάΟΗ ΣέάεάΗπ = ΤάΓδ ΗαΞανψάνάδ αΗ εάάγψρ ζαΗ εφάγσάγυ


εγΗ – ζΡΘί – γδ ήάΟ ΫάΗα έΡΝΚδάΗ = ζΡΪυδάΗ ΪδάζΙπ ΫαγΗδυεάγ ΥάΡγάζΗ


ζΗαΪνάΟυ ανάΣ αγΞαάζής Σάζμ αεγάΗ = ζΪνΟδάΗ δΝάδ ΠίάΡμ Ωαψάα ΗαΝάαυάγυ


έανΡΚΪάΗ ένάε Εδ αάεάζΗπ ζΕδ ΞάάΟσΡΗπ = δΝδ ΗαΤγζΣ ΗαΚν ΚυΜαάμ ΘεάΗ ΗαΩψαάγυ


ζήάΟ ΚΜεά ( ΕΘΡΗεάνάγ ) γγΚΤάήάΗπ = έΓΣάΗπ ανυΝϊΨσάγσ έάν ΓΪνάΗΟε ΗαΥψάδάγυ


γάΗ Θνάδ ΘΫάΟΗΟ ζΗαΓήΥάμ γΤΗΪΡδάΗ = ΚζψΪάΚ ζΣΘάΗδάΗ Ηαάεάγψ ζΗαΓαάάγυ


αίάδψάε ΗαΓγάάα ΗαγάζΪάζΟ ΝάΡψΡδάΗ = ζΗΣΚΘΤΡΚ ΘΗαΝνάΗ ΗαΡζΦάΗΚ ζΗαΓίάγπ


ΡΫγ ΗαΜΡΗΝΗΚ νάΗ (γΝγάζΟυ ) ΚδεΤδάΗ = ΕδάΗ – ζΡΘάάί – ΚΟγνάδάΗ ζδΘΚάΣάγυ


ζαάζ έΡΝδάΗ έάΠΗ έάΓαρ νΜάζ ΘάδάΗ = γΟΗΖάδσ ΗαΝάδφ ΕγψάΗ ΗΜΚΗαδάΗ ΗαάΣάΓγ


ΕδψΗ αδΣΪάΟυ έάν ΗαΓΪνάΗΟφ ΡΫάγ ΓΣάμπ = νήΚΗΚδάΗ ΅ ζήάαάζΘς εάΟψεάΗ ΗαΣάήάγυ


ζγάΗ ΘΡΝδάΗ ΜάζΗΡ ΗαΘνάΚ δίΚΘάεάΗ = Καί ΗαζΥΗνΗ ΗαΚν ΓζΥάμ ΘεάΗ ΗαΝάΡγυ


γΝάγψάαΗΚς ΘΤζής΅ΓαΘάΣάΚ ΓγάάάαΗπ = αγάδ γΚΗΪεάγυ ααήάΟΣ ήάΟ ΝάγάζΗ


ζγΠ ΪήαδΗ – ζΡΘψφ ΗαΪΡΤ- γάΗ ΣίδάΚ = γάδάΗ ΗαήάαάζΘ ζΒανάδάΗ ασδσδϊάΚφάήυάγυ


ζΠίάΡ γΣΜΟδάΗ ΗαΓήάΥάμ γαΗγάδάΗ = ζΫΗνάΙ ΗαήΥάΟ Γδ ΚυΣϊΚσδϊέσάΡσ Ηαεάγάγ


Οζψμ ΗαδένάΡ ζδΝάδ ΗαΜδάΟυ δΪαδάεάΗ = ααΞΗδΪνδ ζγάδ ΞΗδάζΗ ζγάδ ζεφγάζΗ


ζΗαέΜάΡ νάΗ ΣνάΟν αΗΝάΚ ΘΤΗΖάΡευ = ζΪνάΟυ δΥάΡς Θάε ΚΣΚΘΤάΡυ ΗαΓγάάγυ


ΘΗαδάΗΡ δίάζμ ζδΝάδυ ΗαδάζΡσ δΝγαυάε = ΓΘΪάΟσ ΠΗ αάζ ΘσΣσγϊδσάΗ Σάζέσ δυΚψσεσάγυΏ!
