ΓΫδνΙ ααΪζΟΙ
έάαάΣάΨάνάδυ ‘ νάΗ Γγνσ ΗαάΝΗδάνΙ = ζγάεάΟ ΗαΓΤάΗζΣ ‘ ΓΜάΟΗΟνάσάάεϊ
ΡΦάΪΚ εάζΗί ‘ ζανάΟΗ ‘ ζαγ = ΚάέάΗΡήϊ ΡΘάάζΪάί ΓΝάαΗγάνάάάάε
ζΓΪάΤάή έάσζϊΝσ ΛΡΗί ΗαάΝάνάδσ = ΨάάάεάάάζΡ νάάΘάάΗΡίάάε ΡΘάψάνάσάάε
ΓΝάάάδψ Εαάνάί ζΣάνάέυ Ηαάδάάψζμ = νάγάή ‘ γάδψάνσ ‘ ΓΝάάΤάάΗΖάνάάε
ζδάΗΡ ΗαάέάΡΗή ΚΔΜάΜ ΡζΝάάν = έάνάΓΣμ έάΔΗΟν Ϊάαμ ΡζΝάάνάε
ζΓζΡΛ υΤάζήάν ζαάζΪάΙ ήάαΘν = ζΝυάΡήάάΙ δάέάάΣάν αΓΝάάέάάΗΟνάάε
ΝάάΘάάνάάΘάάΙ ήάάαάΘάνσ νάΗ ΫάΗαάνάΙ = ΚάάάήάάάΟΣ ΣάάΡψί νάάΗ Γγψάνάάάάάάσε
**********
ΣάάΓαάάΚ ΟΡζΘάσάίφ ζΗαάάΡψΗΘάνάάάΙ = ΟγάΗΑυ ΗαάΤάεάάάνάΟφ ΘάεάΗ ήάΗδάνάάΙ
ΓαΗ ΚάΠίάάΡνάδσ ΪάεάζΟσ ΗαάΡψΞάΗΑφ = ζΤάγάΣυ ΗαάεάδάΗΑφ αάδάΗ ΡΗΪάνάΙΏ
ζεάα ΚάΠίΡνάδ ΜγΗασ ΗαεΦΗΘφ = ζΞάυΦάάΡΙ Σάεάαφάίφ ζΗαΓζΟνάάΙΏ
ζΡΗΖάΝάΙ ΗαάάΘάΡΚάήάάΗα ΗαάαάΠνάΠφ = ζΚάΫάΡνάΟσ ΓΨάνάΗΡίφ ΗαάΗεάάνάάΙΏ
ζΤάΨάάψίφ αάγάψΗ νάσΝάνάδυ ΗαΓΥναυ = ΪΡζΣΗπ ΘάΝάαάψάΚάεάΗ ΗαάΘάΗεάνάάάάΙΏ
ΓΜάΗΘάΚ ζέάν ΝφάΣψάεάΗ ΫάσΥάάψΙ ρ = ΚάέάΨάψΡσ γάδ ΜάσΡϊΣάεάΗ ήάαάΘάνάάε
ΓΚάΪάάαγ Γδάψνσ γάάδάΠ ΗΫάάΚάυΥάΘάά = Κυ Κάαάα δάΘάΦρ ΘάΞάέάψΗήάνάε Ώ
έάγΗ ΪΟΚυ ΓΪάΡέυ ΫνΡ ΗαΩαΗγ φ = ζΫάάνάάΡσ γσάάΥάάΗΡΪ φΓΘάδάΗΖάνάε Ώ
**********
ΝάΘάάνάάΘάΙ ήάάαάΘάνσ νάΗ ΫάάΗαάάάνάΙ = έάΟΚάίφ δάέάζΣρ ‘ αάδάΗ‘ ΪάΗαάνάάΙ
γάΚάμ νάδΜαν Ϊάδί αναυ ΗαΓΣμ = ζΚάεάΡυ έάνάίφ Ηαάγυδμ ΗαΣΗγνΙΏ
γάΥάΗΘυάίφ ΓΝάΡήσ δέάΣσ ΗαΤψΜνψ = ζδάάέάάΣυ ΗαΓΘάνψ Θάάε ΥάάΗαάάνάΙ
έάνάΗ αάνάΚάδν ΗΣάΚάΟΡψυ ΗαάΣάΝΗΘ σ = ΓΘάΡψΟυ ΓζΜάΗΪσάάίφ ΗαάάΝάάΗγάάνάάάΙ
αΓΣάήάνσ ήάάαάΘάΗπ ίάζΗε ΗαάΜάάζμ = έάάΓΡζν ΟγΗΑπ Θάε ΥάΗΟνάάάΙ
**********
ΝάάΘάάνάάάΘάΙ ήάαάΘάνσ νάάΗ ΫάΗαάνάάΙ = ΚάΫάάδάψΚ ΘάΝυάάάάΘψφάίφ ΓΤάάζΗήάνάάε
ΚάΪάΗαάσνϊ έάαάΣάΨάνδυ ‘ ΝΚμ ΚΡσνϊ = ΝάνάάΗΙ ΗαάγάΞάνάγ ίάάνάέ εφάάνσάε Ώ
νυάΨΗαΪϊάί φήάΟψ Μάγάνάαυ ΗαάήάάσάάζΗγ = ζΘάάάάΠαάάάΚάάυάευ ΡΛάψάάΙ ρΘάάάΗαάάάνάάΙ
ζΜάΣγρ δάΝάάνάαρ ΘάΡΗε ΗαάΣάψήάάΗγυ = ζδάάέάάΣρ αάάάε ΝάάάΡΙ ρ ΒΘάάάνάάάΙ
ζΛίαμ‘ ζΜζΪμ‘ ζΘδΚυ ΤενΟς = ΚάΚάζήυ Εαάμ ΗαάάαάγάΣάΙ φΗαάΝΗδνΙ
υνάΣάΗΖάαί φΗαάΥάΘάνΙ ΗαΝΗΖάάάΡζδσ = Ϊάδ ΗαΪάζΟΙ ΗαΝΡΙ ΗαάΡΗΦάνάΙ Ώ
ζεάα γάδ γάΒΘς αάΡσΝάΘφ ΗαΟνΗΡ = ζεάα ΚάυζΓΟυ ΗαάΫάΡΘΙ ΗαάήΗΣάνάΙ Ώ
**********
Κάήάζα έάαάΣάΨάνάδ : ΓΘάάάδάΗΖάνάε ‘ = ΞάΠζΗ γδ ΤέΗενσ Πν ΗαΓΫδνΙ
νΗ ΩΡνέσ ΗαΨζα ζήψάψέϊ ΚΗήζααί
ΨΗαΚφ ΗαΫΡΘΙ ζΘαΗΟί ΗΝΣδ αάί

ζΚΪΗΤΡ ΗαΫάνΡ ζ ΚάδΣμ ΝάΘάδάάΗ
**********
ΝάΤάΟΚυ έάαάΣάΨάνάδυ ‘ ΒγάΗαάνΙ = έάάάΓαάέάνάΚάεάΗ ίάαάεάάΗ ΫάάΗαάάνάάάΙ
ζΝφάάάΡΚυ ΘάάΓνάάψΚάεάΗ ΓδάάΚάάΤάν = έάίάαάψί ‘ Γγάψνσ ‘ αάάν ΓγάδάάάάνάάΙ
ζαίάδψ ‘ γάδάεάΗ ‘ ΨΫΚ ΡΫάΘάΙ = ΚάυάάΡδάάάψγ ΥάάζΚάάν ζγάάζψΗαάνάάε
έάΪάδάΟ ζΟΗΪνσ εάΠν ΗαάΝάνάάάΗΙ σ = ΚάάΦάάγψ ΡζΗΘάάνάί φ ΜυάάΛάγάΗδάνάε
