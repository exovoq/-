е Э : ц хц уоик Юнэн
е Э Э / е Э е : уоикЧф ицэсЧф
е Э Э е / е Э е е : уоикЧф дЯэЯЧ Чсијцс

сасп - УЮЪэ вхбЩ - кфЯуЧ осЪѕ Хф ( е Э е ) экЪШб ицэсЧ нсУфх эдЪус ксь еЧЦЪ оеэб нэ фѓцѓЧЪіх ( Э ) ШѓкЯх еЧуЪ оеэб ( е ) нэ ЬвЦх ЧсУЮэбЁ ц хц ицэс ШЧсуоЧбфЩ ук Чсуоик ЧсЮнэн .
УуЧ уЧ бувЪі Хсэх Ші ( е Э Э е ) ц (е Э е е ) нхц уоикЧф дЯэЯЧ Чсијцс ц сэгЧ ицэсэњф :
- сЧЭЪцЧС ( е Э Э е ) ксь еЧЦЪ ицэс нэ ЧсфцЧЩ ( Э Э )
- ц ЧдЪуЧс (е Э е е ) ксь еЧуЪ ицэс ( е е ) нэ ЧсЬвС ЧсУЮэб.
