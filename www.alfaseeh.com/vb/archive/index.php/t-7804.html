УцсЧ : еэЧлЩ Чснкс ЧсужЧбк :

Чснкс ЧсужЧбк (эдЯц ), нэпцф ЧсуЧжэ ( дЯЧ ), сУф Уес ЧсУсн цЧцЧ, нЪпЪШ УснЧ ууЯцЯЩ.

нэ Чснкс Шпь, пЪШЪ УснЧ уоецбЩ сУф УесхЧ эЧС, ( эШпэ).

ЫЧфэЧ : еэЧлЩ ЧсуеЯб :

ннэ ЧсуЫЧс ЧсуЪоЯу - эдЯц - ЧсуеЯб ( дЯц ), нЪпцф Усн Чснкс ууЯцЯЩ.

гкэ - нксхЧ гкь - пЪШЪ УснЧ уоецбЩ сУф УесхЧ эЧС.

ЫЧсЫЧ : вэЧЯЩ жуэб бнк уЪес ШЧснкс :

ннэ Чснкс бЬцЪ, Хф Уес ЧсУсн хэ ЧсцЧц, нэпЪШ уЧжэхЧ : бЬЧ.


цнэ Чснкс ( ЬбэЪ ), Ъпцф ЧсУсн уоецбЩ - Ьбь - сУф Уф Уес ЧсУсн хфЧ ЧсэЧС.
