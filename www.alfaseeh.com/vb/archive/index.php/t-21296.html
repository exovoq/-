لا يشترط أن يكون هناك فرق بينهما، كما في ( فرض الله وافترض الله)
ولكن قد تجد أحيانا بعض الفروق مثل:
أولا: ( افتعل ) مطاوع ( فعل )، مثل: سقى واستقى، هدى واهتدى
ثانيا: ( افتعل ) فيها تكلف وتصنع، مثل: فعل وافتعل
ثالثا: ( افتعل ) ضد ( فعل )، مثل: كسب واكتسب، قال تعالى: { لها ما كسبت وعليها ما اكتسبت }
