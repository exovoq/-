Ъбнк : нкс ужЧбк убнцк цксЧуЩ бнкх ЧсжуЩ Ё цЧснЧкс : жуэб угЪЪб ЬцЧвЧ ЪоЯэбх : хц .
цЬусЩ ( Ъбнк ) ЧШЪЯЧЦэЩ сЧ уЭс схЧ уф ЧссХкбЧШ .
цЪфеШ : ЧсцЧц : Эбн кин : Эбн уШфэ ксь ЧснЪЭ Ё сЧ уЭс сх уф ЧсХкбЧШ .
ЪфеШ : нкс ужЧбк убнцк цксЧуЩ бнкх ЧсжуЩ ЧсйЧхбЩ ксь ТЮбх .
пЧф : нЧкс сЪфеШ убнцк цксЧуЩ бнкх ЧсжуЩ ЧсйЧхбЩ ксь ТЮбх .
цЬусЩ ( ЪфеШ ) укицнЩ ксь ( Ъбнк ) сЧ уЭс схЧ уф ЧсХкбЧШ К сУфхЧ укицнЩ ксь ЬусЩ ЧШЪЯЦэЩ .
( цЧскпг еЭэЭ ) Уф эоЧс : Ъбнк : нкс Ё цнЧксх : ( пЧф )
ЪфеШ : нкс цнЧксх : жуэб угЪЪб ЬцЧвЧ ЪоЯэбх : хц .
